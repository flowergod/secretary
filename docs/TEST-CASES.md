## 8. 集成测试场景

### 8.1 完整创建流程

**场景**: 用户说 "周五下午3点带孩子去打篮球"

**预期流程**:
1. NL Parser 解析为 create 动作
2. 提取: title="带孩子去打篮球", start_date=下周五, start_time="15:00"
3. CalendarSuggestion 识别家庭分类 (孩子/篮球)
4. Validation 检查分组和优先级
5. ScheduleService 创建事件
6. Feishu 创建记录
7. iCloud 同步创建事件，返回 UID
8. Feishu 更新记录，写入 icloud_event_id

### 8.2 需要确认的流程

**场景**: 用户说 "下周三下午3点开会"

**预期流程**:
1. NL Parser 解析为 create
2. CalendarSuggestion 返回 needsConfirmation=true
3. 返回确认消息给用户
4. 用户确认后继续创建流程

### 8.3 循环任务创建

**场景**: 用户说 "每2天提醒我锻炼身体"

**预期流程**:
1. NL Parser 识别循环模式
2. is_recurring=true, recurrence_type=weekly
3. 创建周期性任务到 Feishu
